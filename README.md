# 🏠 StayMate Hostel Management System – Frontend

## 🚀 Overview

StayMate is a modern hostel management system that helps manage rooms, residents, billing, maintenance, and more.
This repository contains the **frontend** built using React, Tailwind CSS, and Framer Motion.

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🎬 Framer Motion (animations)
* 🔗 Axios (API calls)
* 🌐 Netlify (Deployment)

---

## 📦 Features

* 🔐 User Authentication (Login/Register)
* 🏠 Room Management UI
* 👥 Resident Management
* 🛠️ Maintenance Requests
* 💳 Billing & Payments UI
* 📄 Invoice View
* ✨ Modern animations and UI

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone <your-frontend-repo-link>
cd frontend
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000
```

### 4️⃣ Run the app

```bash
npm run dev
```
